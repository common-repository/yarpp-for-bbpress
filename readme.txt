YARPP for bbpress enables your forums, posts and replies to appear in YARPP results.  That's it.
== Description ==

YARPP for bbpress is a simple plugin that enables your bbpress content to be included in YARPP results.

Once you install this plugin, simply go to Settings >> Related Posts (YARPP) and forums, topics and 
replies will automatically be included in your "Pool".


== Installation ==

Simple install and you're ready to go!

Note: 

If you're looking to include bbpress results as related posts to other post types (pages, posts, etc), 
you will need to check the box "Display results from all post types" located under "Relatedness" options.
